Editor de Video con Difuminado Manual y Automático de Rostros

Aplicación desarrollada en Python que permite reproducir videos y aplicar difuminado a formas o rostros mediante tecnología de difuminado manual o automático con inteligencia artificial, con el objetivo de proteger la identidad de las personas.

Características:

  • Detección automática de formas y rostros mediante inteligencia artificial con YOLOv8
  • Difuminado manual con seguimiento (tracking) de objetos
  • Control rápido de reproducción de video: play, pausa, velocidad hasta x16, saltos por bloques de frames, frame anterior y siguiente
  • Captura de frames como imágenes para posterior entrenamiento de modelos de IA
  • Guardado del progreso de difuminado en archivos JSON, permitiendo continuar el trabajo sin reprocesar el video
  • Exportación de video utilizando FFmpeg con soporte GPU (DirectML)
  • Aplicación optimizada para equipos de bajo rendimiento, con capacidad de funcionamiento utilizando solo CPU sin necesidad de GPU dedicada

Nota: Para obtener detecciones automáticas más precisas en videos complejos (por ejemplo, cámaras CCTV), el modelo de inteligencia artificial puede requerir la asistencia del desarrollador para el entrenamiento o ajuste utilizando imágenes del video a procesar. En estos casos se recomienda contactar al correo del desarrollador.

Tecnologías: Python, YOLO, OpenCV, PyTorch, FFmpeg

Monitorear Desconexiones de NVR y Cámaras

Proyecto desarrollado en Python para Windows que supervisa múltiples rangos de direcciones IP dentro de una red local, incluyendo dispositivos críticos como NVRs y cámaras de videovigilancia.

El sistema detecta en tiempo real fallas de conectividad y, cuando una IP permanece inaccesible durante un tiempo configurable, envía una alerta automática a través de Telegram.

Implementa monitoreo concurrente para garantizar eficiencia incluso al supervisar decenas de dispositivos simultáneamente.
