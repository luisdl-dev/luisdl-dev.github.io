Este proyecto fue desarrollado en Python y está diseñado para ejecutarse en entornos Windows. Permite supervisar múltiples rangos de direcciones IP dentro de una red local, incluyendo dispositivos críticos como NVRs y cámaras de videovigilancia. El sistema realiza verificaciones periódicas de conectividad y detecta en tiempo real dispositivos que presentan fallas o pérdida de comunicación. Cuando una dirección IP permanece inaccesible durante un tiempo configurable, se genera automáticamente una alerta mediante la API de Telegram, permitiendo una respuesta rápida ante incidentes. La arquitectura del proyecto está optimizada para monitoreo concurrente, asegurando eficiencia incluso al supervisar decenas de dispositivos simultáneamente.
